How to solve problems



Problem

Suggestion

Connection cannot be setup

If you are behind a proxy you need to configure the http proxy:

java -Dhttp.proxyHost=proxyhost [-Dhttp.proxyPort=portNumber] visualap.VisualAp

Detected collision between two or more output pins

You have to unbind the output pins that have been connected together

Detected a floating input pin

You need to connect the input pin to another output pin

Detected input pins not connected to any output pin

You need to connect the input pins to another output pin

Detected a loop
Detected a cycle

You have to break the loop by reconnecting pins in a different way
